Course Description

This course provides advanced instruction in web development technologies, including server-side programming, database integration, API development, and modern web frameworks. Students will build upon foundational web development skills to create dynamic, data-driven web applications.

Learning Objectives

Upon successful completion of this course, students will be able to:

  • Design and implement server-side web applications using modern frameworks
  • Integrate databases with web applications for data persistence
  • Develop and consume RESTful APIs
  • Apply security best practices in web development
  • Deploy web applications to cloud platforms
  • Utilize version control systems for collaborative development

Required Materials

  • Textbook: "Modern Web Development" (3rd Edition) - Available through course reserves
  • Access to a computer with internet connection
  • GitHub account (free)
  • Code editor (VS Code recommended - free)

Course Policies

Late Work: Assignments submitted after the deadline will receive a 10% deduction per day, up to 3 days. After 3 days, late work will not be accepted without prior arrangement.

Academic Integrity: Students are expected to complete their own work. Plagiarism or cheating will result in a zero for the assignment and may result in further disciplinary action.

Participation: Active participation in weekly discussions is required. Students should log in at least 3 times per week.
